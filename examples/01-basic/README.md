# 基础示例

本目录包含 SW Runtime 的基础功能演示。

## 文件说明

### 基础语法
- **calculator-app.ts** - 计算器应用示例
- **example-es6.ts** - ES6 语法示例
- **es6-math.ts** - ES6 数学函数示例
- **math.ts** - TypeScript 数学模块

### 模块系统
- **logger.js** - 日志模块示例
- **utils.js** - 工具函数模块
- **test_module.js** - 测试模块

### 综合示例
- **comprehensive-demo.ts** - 综合功能演示
- **config.json** - 配置文件

## 运行示例

```bash
# 运行计算器应用
sw_runtime run examples/01-basic/calculator-app.ts

# 运行 ES6 示例
sw_runtime run examples/01-basic/example-es6.ts

# 运行综合演示
sw_runtime run examples/01-basic/comprehensive-demo.ts
```

## 功能特点

- ✅ TypeScript 支持
- ✅ ES6+ 语法
- ✅ 模块导入/导出
- ✅ CommonJS 和 ES6 模块
- ✅ JSON 配置文件
