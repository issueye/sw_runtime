文件系统模块 (fs)

fs

提供完整的文件系统操作功能，包括同步和异步方法。

fs.readFileSync(filename: string, encoding?: string): string
同步
同步读取文件内容。
参数
filename string 必需
文件路径
encoding string 可选
文件编码，默认为 'utf8'
const fs = require('fs'); // 读取文本文件 const content = fs.readFileSync('config.json', 'utf8'); console.log(content); // 读取并解析 JSON 配置 const configText = fs.readFileSync('config.json'); const config = JSON.parse(configText); console.log('配置:', config);
fs.writeFileSync(filename: string, data: string): void
同步
同步写入文件。
// 写入文本文件 fs.writeFileSync('output.txt', 'Hello World'); // 写入 JSON 数据 const data = { name: '张三', age: 25 }; fs.writeFileSync('user.json', JSON.stringify(data, null, 2)); // 写入日志 const logEntry = `[${new Date().toISOString()}] 应用启动\n`; fs.writeFileSync('app.log', logEntry);
fs.readFile(filename: string): Promise<string>
异步 Promise
异步读取文件内容。
// 使用 Promise fs.readFile('data.txt') .then(content => { console.log('文件内容:', content); }) .catch(error => { console.error('读取失败:', error); }); // 使用 async/await async function readData() { try { const content = await fs.readFile('data.txt'); console.log('文件内容:', content); } catch (error) { console.error('读取失败:', error); } }
fs.existsSync(path: string): boolean
同步
检查文件或目录是否存在。
// 检查文件是否存在 if (fs.existsSync('config.json')) { console.log('配置文件存在'); const config = JSON.parse(fs.readFileSync('config.json')); } else { console.log('配置文件不存在，使用默认配置'); const defaultConfig = { port: 3000, debug: false }; fs.writeFileSync('config.json', JSON.stringify(defaultConfig, null, 2)); } // 检查目录是否存在 if (!fs.existsSync('logs')) { fs.mkdirSync('logs'); console.log('创建日志目录'); }
fs.mkdirSync(path: string, options?: {recursive?: boolean}): void
同步
创建目录。
// 创建单级目录 fs.mkdirSync('logs'); // 递归创建多级目录 fs.mkdirSync('data/cache/temp', { recursive: true }); // 创建项目结构 const dirs = ['src', 'dist', 'logs', 'config']; dirs.forEach(dir => { if (!fs.existsSync(dir)) { fs.mkdirSync(dir); console.log(`创建目录: ${dir}`); } });
