路径操作模块 (path)

path

提供跨平台的路径操作功能，兼容不同操作系统的路径格式。

path.join(...paths: string[]): string
同步
连接多个路径片段。
参数
...paths string[] 必需
要连接的路径片段
const path = require('path'); // 连接路径 const fullPath = path.join('home', 'user', 'documents', 'file.txt'); console.log(fullPath); // home/user/documents/file.txt // 处理相对路径 const relativePath = path.join('src', '..', 'dist', 'index.js'); console.log(relativePath); // dist/index.js
path.resolve(...paths: string[]): string
同步
解析为绝对路径。
// 解析为绝对路径 const absolutePath = path.resolve('src', 'utils', 'helper.js'); console.log(absolutePath); // /current/working/directory/src/utils/helper.js // 从根目录开始 const fromRoot = path.resolve('/usr', 'local', 'bin'); console.log(fromRoot); // /usr/local/bin
path.dirname(path: string): string
同步
获取路径的目录部分。
const filePath = '/home/user/documents/report.pdf'; const directory = path.dirname(filePath); console.log(directory); // /home/user/documents
path.basename(path: string, ext?: string): string
同步
获取路径的文件名部分。
参数
path string 必需
文件路径
ext string 可选
要移除的扩展名
const filePath = '/home/user/documents/report.pdf'; // 获取完整文件名 const filename = path.basename(filePath); console.log(filename); // report.pdf // 获取不带扩展名的文件名 const nameOnly = path.basename(filePath, '.pdf'); console.log(nameOnly); // report
path.extname(path: string): string
同步
获取路径的扩展名。
const files = [ 'document.pdf', 'script.js', 'style.css', 'image.png', 'README' ]; files.forEach(file => { const ext = path.extname(file); console.log(`${file} -> ${ext || '(无扩展名)'}`); }); // 输出: // document.pdf -> .pdf // script.js -> .js // style.css -> .css // image.png -> .png // README -> (无扩展名)
path.isAbsolute(path: string): boolean
同步
判断路径是否为绝对路径。
const paths = [ '/usr/local/bin', './relative/path', '../parent/dir', 'C:\\Windows\\System32', 'relative/path' ]; paths.forEach(p => { const isAbs = path.isAbsolute(p); console.log(`${p} -> ${isAbs ? '绝对路径' : '相对路径'}`); });
path.relative(from: string, to: string): string
同步
计算从一个路径到另一个路径的相对路径。
const from = '/home/user/projects/app'; const to = '/home/user/projects/shared/utils'; const relativePath = path.relative(from, to); console.log(relativePath); // ../shared/utils // 反向计算 const backPath = path.relative(to, from); console.log(backPath); // ../../app
path.normalize(path: string): string
同步
规范化路径，解析 '.' 和 '..' 片段。
const messyPaths = [ '/home/user/../user/./documents', 'src//utils//../helpers/./index.js', './a/b/c/../../d' ]; messyPaths.forEach(p => { const normalized = path.normalize(p); console.log(`${p} -> ${normalized}`); }); // 输出: // /home/user/../user/./documents -> /home/user/documents // src//utils//../helpers/./index.js -> src/helpers/index.js // ./a/b/c/../../d -> a/d
路径常量
提供平台相关的路径常量。
// 路径分隔符 console.log('路径分隔符:', path.sep); // Windows: \, Unix: / // 环境变量分隔符 console.log('环境变量分隔符:', path.delimiter); // Windows: ;, Unix: : // 构建跨平台路径 const crossPlatformPath = ['home', 'user', 'documents'].join(path.sep); console.log(crossPlatformPath);
