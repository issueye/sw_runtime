加密模块 (crypto)

crypto

提供完整的加密功能，包括哈希、编解码、对称加密和随机数生成。

crypto.sha256(data: string): string
同步
计算字符串的 SHA256 哈希值。
参数
data string 必需
要计算哈希的字符串
返回值
string - 十六进制格式的 SHA256 哈希值
const crypto = require('crypto'); const hash = crypto.sha256('hello world'); console.log(hash); // 输出: b94d27b9934d3e08a52e52d7da7dabfac484efe37a5380ee9088f7ace2efcde9 // 其他哈希函数 const md5Hash = crypto.md5('hello'); const sha1Hash = crypto.sha1('hello'); const sha512Hash = crypto.sha512('hello');
crypto.base64Encode(data: string): string
同步
将字符串编码为 Base64 格式。
const encoded = crypto.base64Encode('hello world'); console.log(encoded); // aGVsbG8gd29ybGQ= const decoded = crypto.base64Decode(encoded); console.log(decoded); // hello world
crypto.aesEncrypt(data: string, key: string): string
同步
使用 AES-256-GCM 算法加密数据。
参数
data string 必需
要加密的数据
key string 必需
加密密钥
const secretData = '这是机密信息'; const secretKey = 'my-secret-key-2023'; // 加密数据 const encrypted = crypto.aesEncrypt(secretData, secretKey); console.log('加密后:', encrypted); // 解密数据 const decrypted = crypto.aesDecrypt(encrypted, secretKey); console.log('解密后:', decrypted); // 这是机密信息
crypto.randomBytes(size?: number): string
同步
生成指定长度的随机字节序列。
参数
size number 可选
字节长度，默认为 16
// 生成默认长度的随机字节 const random16 = crypto.randomBytes(); console.log('16字节随机数:', random16); // 生成32字节的随机字节 const random32 = crypto.randomBytes(32); console.log('32字节随机数:', random32); // 用于生成会话ID const sessionId = crypto.randomBytes(24);
