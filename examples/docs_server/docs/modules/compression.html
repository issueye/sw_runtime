压缩模块 (compression)

compression / zlib

提供 Gzip 和 Zlib 压缩/解压功能，支持高性能数据压缩。

compression.gzipCompress(data: string): string
同步
使用 Gzip 算法压缩数据。
参数
data string 必需
要压缩的字符串数据
返回值
string - Base64 编码的压缩数据
const compression = require('compression'); const originalData = '这是一段需要压缩的长文本数据，包含重复的内容，重复的内容，重复的内容...'; // Gzip 压缩 const compressed = compression.gzipCompress(originalData); const decompressed = compression.gzipDecompress(compressed); console.log('原始大小:', originalData.length); console.log('压缩后大小:', compressed.length); console.log('压缩比:', (compressed.length / originalData.length * 100).toFixed(2) + '%'); console.log('解压正确:', decompressed === originalData);
compression.zlibCompress(data: string): string
同步
使用 Zlib 算法压缩数据。
// Zlib 压缩 const zlibCompressed = compression.zlibCompress(originalData); const zlibDecompressed = compression.zlibDecompress(zlibCompressed); console.log('Zlib 压缩后:', zlibCompressed.length); console.log('Zlib 解压正确:', zlibDecompressed === originalData);
