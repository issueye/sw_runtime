运行时概述

关于 SW Runtime

SW Runtime 是一个基于 Go 和 goja 的高性能 JavaScript/TypeScript 运行时，提供了完整的模块系统和丰富的内置功能模块，专为企业级应用设计。

核心特性
SW Runtime 提供以下核心特性：
  • 模块系统: 支持 CommonJS require() 和 ES6 动态 import()
  • TypeScript 支持: 自动编译 .ts 文件，无需额外配置
  • 异步支持: 完整的 Promise 和事件循环支持
  • 内置模块: 加密、压缩、文件系统、HTTP、数据库等
  • 高性能: 基于 Go 的高并发和低内存占用
  • 跨平台: 支持 Windows、Linux、macOS
